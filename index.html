📋 How to set up Google Sheets (2 minutes)
1

Create a new Google Sheet

Go to sheets.google.com → create a new blank spreadsheet → name it Ace Life Tracker.

2

Open Apps Script

In the sheet, click Extensions → Apps Script. Delete any existing code. Paste the entire script below:


      
    


    
3

      
Enable Drive API in Apps Script

      
In Apps Script, click Services (the + icon on the left sidebar) → find Drive API → click Add. This lets the script do OCR for free using Google Drive.

    


    
4

      
Deploy as Web App

      
Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy. Copy the Web App URL.

    


    
4

      
Paste URL above & test

      
Paste the URL in the field above and click Test connection. You'll see ✅ if it's working. That's it — your data now syncs to Google Sheets automatically.

    


    
5

      
Share the sheet with your family

      
In Google Sheets, click Share and add family members' Gmail addresses. They can view all sales data directly in the spreadsheet — no app needed.
