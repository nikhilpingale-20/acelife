Setup steps

    

      
1
Create a Google Sheet
Go to sheets.google.com → new blank spreadsheet → name it Ace Life Tracker.

      
2
Paste the script
In the sheet → Extensions → Apps Script → delete all existing code → paste the script above → Ctrl+S to save.

      
3
Deploy as Web App
Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy → Authorize → copy the URL.

      
4
Paste URL & test
Paste the URL in the field above → click Test connection.

      
5
Share dashboard with others
Share the dashboard.html URL with your family/team. They get view-only access. Only this admin page can upload bills and edit products.
